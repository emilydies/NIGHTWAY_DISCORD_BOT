# Nightway Discord Bot

Бот для управления приватными голосовыми каналами и ролями на Discord сервере.

## Основные функции
- **Приватные комнаты**: Автоматическое создание временных голосовых каналов.
- **Управление комнатой**: Команды для смены названия, лимита пользователей, закрытия/открытия доступа и передачи владельца.
- **Панель выбора ролей**: Интерактивная панель для выбора языка (RU/ENG) через реакции.
- **Авто-удаление**: Пустые временные каналы удаляются сами.

## Требования
- [Node.js](https://nodejs.org/) (версия 16.9.0 или выше)
- Токен бота в [Discord Developer Portal](https://discord.com/developers/applications)

## Установка

1. Склонируйте репозиторий.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Создайте файл `.env` в корневой папке и добавьте настройки:
   ```env
   TOKEN=ваш_токен_бота
   GUILD_ID=id_вашего_сервера
   # Опционально:
   ROLE_RU_NAME=RU
   ROLE_ENG_NAME=ENG
   CREATE_VC_NAME=Create VC
   VC_CATEGORY_NAME=Voice
   REACTION_CHANNEL_NAME=roles
   ```

## Запуск

### Обычный запуск:
```bash
node index.js
```

### Запуск в режиме разработки (авто-перезагрузка):
```bash
npx nodemon index.js
```

## Список команд (префикс `!`)
- `!help` — Показать все доступные команды.
- `!setuproles` — Создать панель выбора ролей (админская).
- `!myvc` — Информация о вашей приватной комнате.
- `!name <название>` — Переименовать комнату.
- `!limit <0-99>` — Установить лимит участников.
- `!lock` / `!unlock` — Закрыть или открыть комнату для всех.
- `!hide` / `!show` — Скрыть или показать комнату в списке.
- `!invite @user` / `!uninvite @user` — Управление доступом конкретных пользователей.
- `!kick @user` — Выгнать пользователя из вашей комнаты.
- `!transfer @user` — Передать права владельца комнаты.
- `!claim` — Забрать владение комнатой, если текущий владелец вышел.
